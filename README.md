# html-tourguide
Generateoffline HTML travel guides from any itinerary.

This skill helps people who suffer from multi-app tour route plan,Instead of switching between notes, map apps, screenshots, and travel documents, this skill turns your travel plan into a mobile-friendly HTML guide that works as a lightweight travel companion.

## when to use this skill

Use this skill when you want to:
- organize a travel itinerary into a clean guide
- build a city walk guide
- create a themed travel experience
- share your itinerary with friends or family
- export a travel plan as a single HTML file
- avoid constantly switching between Notes and Map apps

## What this Skill does
This skill can:
- Design or refine a travel itinerary
- Organize destinations into a logical route
- Generate destination descriptions
- Create a mobile-friendly HTML guide
- Add map links for each destination
- Produce a guide that can be shared or opened offline

## Why HTML?
HTML offers several advantages over PDF or Markdown:
- Responsive on mobile devices
- Interactive navigation
- Clickable map links
- Easy to share
- No installation required
- Works offline after download
